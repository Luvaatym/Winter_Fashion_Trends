# Анализ данных зимней моды

## Описание проекта
Данный проект посвящён анализу данных о зимней моде с использованием языка программирования Python и библиотек анализа данных, а также платформы визуальной аналитики Yandex DataLens.

## Цель проекта
Целью проекта является изучение структуры датасета, оценка качества данных и выявление основных закономерностей, связанных с характеристиками зимней одежды, включая категории товаров, материалы, ценовые диапазоны, популярность и сезонность.

## Результаты
В ходе анализа были изучены распределения товаров по категориям и сезонам, определены наиболее популярные виды зимней одежды и бренды, а также проанализирована зависимость популярности товаров от их цены и характеристик.
